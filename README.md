# WB Manager Dashboard

Онлайн **РНП** для менеджеров WB: куда смотреть, STR, воронка, отгрузка, реклама.

## Ссылка

**https://wedoubledi-web.github.io/wb-logistics-dashboard/**

## Вкладки

| Вкладка | Для чего |
|---------|----------|
| **РНП** | Пульс кабинета: неделя vs неделя, выручка, заказы, ДРР, выкуп |
| **Фокус сегодня** | Куда смотреть прямо сейчас + инсайты по правилам |
| **STR** | Просели — продажи W1 vs W0, что разбирать |
| **Воронка** | Выкуп %, возвраты, средний чек по SKU |
| **Отгрузка** | План закупки/отгрузки по географии |
| **Логистика** | ИЛ, экономия, стоимость логистики |
| **Реклама** | ДРР, стоп/срез РК |
| **Остатки** | Склады WB, оборачиваемость |
| **Ярд 2.0** | Зоны, чек-лист менеджера |
| **Склады** | География спроса, план |
| **Стоп** | Не грузить, минусовые |

## Обновление

```bash
python3 Бизнес/Логистика/scripts/wb_manager_fetch.py
python3 Бизнес/Логистика/scripts/publish_dashboard.py
```

Источники:
- `wb-supply` — продажи, остатки, география
- `daily-brief` — заказы, выкупы, реклама, финотчёт, инсайты

## Локально

```bash
python3 Бизнес/Логистика/scripts/wb_manager_fetch.py --rebuild-only
python3 Бизнес/Логистика/Дашборд/serve.py
```
